Support

Need help? Here are the fastest ways to troubleshoot and report issues.

Report an issue (recommended)

Use the in-extension “Report an issue” button. It opens a report page that can generate anonymized diagnostics (no prompt/response text).

Common fixes

  • Buttons not showing: refresh the LLM page and make sure the extension has access to the site.
  • Reveal/Restore not working after a while: the local per-tab vault expires automatically; re-run Secure Send.
  • False positive warning: you can continue unprotected once, or use Secure Send to sanitize.
