Features that make LLM work safer.

SafePrompt is designed to be simple: it warns you before risky sends, sanitizes with Secure Send, and restores locally when you choose.

Coverage (grouped)

High-signal categories that matter for SMB workflows — without overwhelming you with an endless list.

PII

Emails, phone numbers (US/IL), IDs (multiple regions), dates of birth (context-aware warnings), select address patterns.

Secrets

JWTs, API keys, GitHub/Slack tokens, Stripe secrets, and private key blocks.

High‑risk identifiers

Credit cards (validated), IBAN (checksum), SWIFT/BIC (keyword), IP/MAC, GPS, wallet-like addresses.

Workflow features

Designed to fit how teams actually use LLMs day to day.

Warn before send (free)

If you’re about to send sensitive info, SafePrompt can warn you and suggest Secure Send. Warn is always free.

Secure Send (sanitize)

Replaces matched values with tokens like [[SP_EMAIL_1]] so sensitive data doesn’t leave your device.

Reveal/Restore

Swap tokens back locally when you need to review the answer in full context.
